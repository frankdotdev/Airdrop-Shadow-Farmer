# Shadow Farmer

A Telegram Mini App for automated airdrop farming in Web3, designed to mimic human behavior and bypass detection.

## Features

- Non-custodial automation using WalletConnect
- Human-like randomization to avoid bans
- Multi-chain support (EVM, Solana)
- Sybil score checker
- Gas guard
- Auto CAPTCHA solving
- Subscription tiers

## Setup

1. Clone the repo
2. Install dependencies: `poetry install`
3. Set up .env from .env.example
4. Run migrations: `alembic upgrade head`
5. Start backend: `uvicorn app.main:app --reload`
6. Start frontend: `cd frontend && npm run dev`
7. Deploy with Docker: `docker-compose up`

## Deployment

Use Docker Compose for easy deployment on Render, Railway, etc.

## Security

- AES-256 encryption for sessions
- Residential proxies
- Fingerprint spoofing
