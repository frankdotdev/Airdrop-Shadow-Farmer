# User Guide: How to Use Shadow Farmer

This guide explains how to get started and use Shadow Farmer as a user.

## Getting Started

### 1. Access the App
- Open Telegram and search for @AirdropShadowFarmerBot or use the Mini App link: [t.me/airdropshadowfarmerbot].
- Click "Start" to launch the Mini App.

### 2. Sign Up
- Authenticate with your Telegram ID (no email or password needed).
- Generate a referral code (e.g., SF12345) – share to earn commissions.
- Choose tier:
  - **Hobbyist**: Free, 1 wallet, basic farming.
  - **Whales**: $10/month, unlimited wallets, advanced strategies.
  - **Protocol**: $50/month, priority, custom support.

### 3. Add a Wallet
- Click "Wallets" in the sidebar.
- Connect via WalletConnect: Scan QR or use deep link (MetaMask, Trust Wallet, Phantom for Solana).
- Enter wallet address (e.g., 0x... for EVM, Sol... for Solana).
- App assigns proxy and calculates Sybil score (aim for <20 for best results).
- Tip: Use fresh wallets for higher success.

### 4. Dashboard Overview
- **Bento Grid**: Quick stats – Total Profit, Active Tasks, Sybil Score, Recent Airdrops.
- **Activity Feed**: Real-time updates on farming tasks (pending, running, completed).
- **Profit Tracker**: Estimated USD value from farmed tokens (via Coingecko).

### 5. Farming Strategies
- Navigate to "Strategies".
- Select type:
  - **Bridge**: Transfer ETH to Base for airdrops (sliders for amount, aggression).
  - **Swap**: Uniswap/Jupiter swaps for liquidity tasks.
  - **Vote**: Governance votes on protocols.
  - **Health**: Balance checks, small txs to stay active.
- Toggle Gas Guard (auto-retries high gas).
- Set aggression: Low (safe, slow), High (fast, riskier).
- Click "Start Farming" – tasks queue asynchronously.

### 6. Airdrop Scanner
- Go to "Scanner".
- View new opportunities: Project name, chain, potential reward.
- Click "Farm This" to add to strategies.
- Notifications: Telegram alerts for high-potential drops.

### 7. Profile & Subscriptions
- View referral stats, earnings.
- Upgrade tier: Pay with USDT (scan QR, confirm tx).
- Verify payment: App checks on-chain, unlocks features.
- Withdraw commissions: To your wallet.

### 8. Monitoring & Settings
- **Notifications**: Enable Telegram pings for task completion, low balance.
- **Proxies**: Rotate manually if needed.
- **Dark Theme**: Toggle for UI.
- **Copy Addresses**: Click with checkmark animation.

## Tips for Success
- **Start Small**: Test with $10 ETH on testnet.
- **Diversify**: Use multiple wallets/chains.
- **Monitor Gas**: Farm during low-fee times.
- **Avoid Bans**: Keep Sybil low, don't over-farm one protocol.
- **Referrals**: Share code for 10% on subs, tiered rewards (5 referrals = free month).

## Troubleshooting
- **Wallet Connect Fails**: Refresh, check project ID.
- **Task Stuck**: Check queue in Dashboard, restart.
- **CAPTCHA**: Auto-solved, but manual if fails.
- **Payments**: Ensure USDT on correct chain (Ethereum).
- **Support**: /help in bot or email support@shadowfarmer.com.

## FAQ
- **Is it safe?** Yes, non-custodial – you sign all txs.
- **Fees?** Gas + subscription; no hidden.
- **Refunds?** No, but 7-day trial.
- **Chains?** EVM (Eth, Base), Solana; more coming.
